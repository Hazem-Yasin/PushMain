Essential C and C++ Concepts

3. Array Basics

What are Arrays?

A collection of similar data elements grouped under one name.

int main() {
    // This is an array of 5 integers
    int A[5];
}

Memory Layout:

Heap 
A[5]
Code Section:
int main() {
    int A[5];
}

Declaring vs Initializing an Array

// Declaring an array
int A[5];

// Initializing an array
int A[5] = {0,1,2,3,4};

Accessing an Array

for(int i = 0; i < 5; i++) {
    printf("%d", A[i]);
}

5. Structures

What is a Structure?

A collection of different types of data grouped under one name.

struct Rectangle {
    int length;
    int breadth;
};

Accessing structure members:

Rectangle r;
r.length = 10;
printf("Length: %d", r.length);

Array of Structures

struct Card {
    int face;
    int shape;
    int color;
};

Card deck[52];

7. Pointers

What is a Pointer?

A variable that stores the address of another variable.

int a = 10;
int *p = &a;

Accessing a pointer:

printf("%d", *p); // Outputs the value stored in a

9. Reference in C++

What is a Reference?

A reference is another name for an existing variable (only in C++).

int a = 10;
int &r = a;

11. Pointer to Structure

struct Rectangle {
    int length;
    int breadth;
};

Rectangle r;
Rectangle *p = &r;
p->length = 20;

13. Functions

What are Functions?

A function is a reusable block of code that performs a specific task.

void printMessage() {
    printf("Hello, World!");
}
